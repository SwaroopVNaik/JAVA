# USECASES OF VARIBALES IN JAVA

# Instance Variable => 

## When to use => 

- if variable value is varied from object to object.
- seprate copy will create for every object
- default values based on data object 

##  Accessed =>

- using Object Only 

## where to declare => 

- Inside a class outside method 

# Static Variable =>

- if variable value is not varied from object to object 
- class level 
- only one copy will be created and shared to all object
- default values based on data type 

## Accessed => 

- a) using class name 
- b) using Object 
- c) directly in static context
